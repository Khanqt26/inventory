# BUILDPLAN Document Index

This folder contains planning and deployment documents for the project.

## Quick Start

- If you need to deploy/update online: open `Render_Deployment_and_Data_Sync_Guide.md`.
- If you need final submission/deployment checks: open `Online_Deployment_Checklist.md`.
- If you need proposal/reference content: open the proposal files below.

## File Map

- `Render_Deployment_and_Data_Sync_Guide.md`
  - Step-by-step guide for Render deployment and local data sync to hosted database.
- `Online_Deployment_Checklist.md`
  - Practical checklist for preparing and validating online deployment.
- `Project_Proposal_Sidewalk_Refreshments.md`
  - Project proposal in markdown format.
- `Project_Proposal_Sidewalk_Refreshments.doc`
  - Project proposal in Word document format.
- `buildplan.doc`
  - Original project build plan document.
- `mvp and prd.doc`
  - MVP + PRD reference document.

## Recommended Workflow

1. Update code and test locally.
2. Follow `Render_Deployment_and_Data_Sync_Guide.md` for deploy/data sync.
3. Use `Online_Deployment_Checklist.md` before final submission/demo.
